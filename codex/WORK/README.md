# Work Directory

This directory contains all operational project work artifacts.

Subsections:

- OBJECTIVES.md → ranked active objectives
- TASK_GRAPH.jsonl → append-only canonical task records
- EPICS/ → larger decomposed initiatives
- RUNBOOKS/ → operational procedures

This directory represents the deterministic execution state of the project.

Overseer must treat TASK_GRAPH.jsonl as the source of truth for task lifecycle state.

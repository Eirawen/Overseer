# Constraints

## Human Time Budget
Daily: 6–10 hours, but preserve deep work blocks.

## Compute Budget
V0: no hard cap; log usage optionally in /codex/08_TELEMETRY.

## Interrupt Thresholds
- Maximum interrupts/day (default): 6
- Quiet hours allowed: yes/no (default: no)
- "Emergency interrupt" conditions:
  - task graph blocked > 50%
  - safety/security issue detected
  - repeated recursion failure on core path

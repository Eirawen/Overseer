# Context Stream (Continuity Layer)

Purpose: preserve the "we talked yesterday" continuity.

Rules:
- do NOT dump raw chat logs
- write compressed snapshots, each <= ~300 lines
- capture unresolved tensions, intuition, and recent conversational anchors
